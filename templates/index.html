Overview

Taxpayers API es un servicio REST que facilita la obtención de datos sobre los contribuyentes registrados en la DGII para que otras aplicaciones puedan tener acceso a estos datos. Esta información es recolectada de la plataforma DGII RNC y orginazada en formato JSON para un facil manejo y consumo.

Entre los datos que se pueden obtener están: RNC o Cédula, Razón social, Nombre comercial, estado, fecha de registro, etc.

BASE URL

https://dgii-taxpayers-info.herokuapp.com/api/v1

OJO

Este servicio esta hospedado en un servidor gratuito para fines demostrativos. Esto quiere decir que la url base puede variar y tiene sus limitaciones en cuanto a peticiones http. Pero de igual manera puedes usarlo.

Endpoints

/taxpayers GET

Consultar todos los contribuyentes

Parámetros

limit /taxpayers?limit=3

Este parámetro es para limitar el número de registros. Como es una gran cantidad de registros, por defecto limit tiene como valor 100, o sea que si este parámetro no es especificado en la URL va a retornar 100 registros.

Response

/taxpayers/by-rnc GET

Consultar por RNC o Cédula un contribuyente. También puedes consultar varios contribuyentes, si quieres buscar todos los contribuyentes que tengan alguna secuencia en común en sus RNCs o Cédulas. Ej: consultar todos los contribuyentes que contengan los números 402 en sus RNCs o cédula

Parámetros

search /taxpayers/by-rnc?search=401093657

Este parámetro es obligatorio en este endpoint para poder consultar por RNC o Cédula.

limit /taxpayers/by-rnc?search=402&limit=5

Si vas a consultar varios contribuyentes con alguna secuencia en común en sus RNCs o Cédula y la cantidad de registros es muy grande puede usar el parámetro limit para limitar la cantidad de registros.

/taxpayers/by-name GET

Consultar por nombre un contribuyente. También puedes consultar varios contribuyentes, si quieres buscar todos los contribuyentes que tengan algún parámetro en común en sus nombres. Ej: todos los contribuyentes que sean ferreterías, colocas como valor ferretería y retornará todos los contribuyentes que en su nombre contenga ferretería.

Parámetros

search /taxpayers/by-name?search=ferreteria

Este parámetro es obligatorio en este endpoint para poder consultar por nombre.

limit /taxpayers/by-name?search=ferreteria&limit=2

Si vas a consultar varios contribuyentes con algún parámetro en común y la cantidad de registro es muy grande puede usar el parámetro limit para limitar la cantidad de registros. Por defecto tiene como valor 100.

Response

/taxpayers/active-state GET

Consultar todos los contribuyentes activos.

Parámetros

limit /taxpayers/active-state?limit=2

Puedes usar el parámetro limit para limitar la cantidad de registros. Si no específica este parámetro, por defecto el valor es 100.

Response

/taxpayers/inactive-state GET

Consultar todos los contribuyentes no activos.

Parámetros

limit /taxpayers/inactive-state?limit=2

Puedes usar el parámetro limit para limitar la cantidad de registros. Si no específica este parámetro, por defecto el valor es 100.

Response

/taxpayers/discharged-state GET

Consultar todos los contribuyentes dado de baja.

Parámetros

limit /taxpayers/discharged-state?limit=2

Puedes usar el parámetro limit para limitar la cantidad de registros. Si no específica este parametro, por defecto el valor es 100.

Response
